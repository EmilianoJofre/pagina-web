Bifurcaciones

Las bifurcaciones sirven para crear lapsos de tiempos alternos, con los cuales podemos hacer cambios al proyecto de los cuales hacemos algo completamente distinto pero en paralelo a nuestra version master, sirve como metodo para hacer pruebas y experimentos con el proyecto :)
